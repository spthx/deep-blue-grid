# DEEP BLUE GRID

ゲームボーイ時代のレーダー作戦ゲームを現代的に再構成した、1人用・1ステージ完結のブラウザ海戦ゲームです。8×8の海域へ3隻を配置し、通常砲撃・特殊兵装・レーダーを使って敵AI艦隊の全撃沈を目指します。

## 起動方法

Node.js 22.13 以上を用意し、プロジェクトのフォルダで次を実行します。

```bash
npm install
npm run dev
```

ターミナルに表示されたURLをPCまたはスマートフォンのブラウザで開いてください。同じWi-Fi内のスマートフォンで試す場合は、必要に応じて `npm run dev -- --host 0.0.0.0` を使います。

本番ビルドと自動テスト:

```bash
npm run build
npm test
```

## 操作

- 配置: 艦艇カードを選び海図をクリックまたはタップ
- 回転: 右クリック、または `R`
- 照準移動: 矢印キー / WASD / ポインター
- 決定: Enter / Space / タップ
- コマンド: `1` FIRE、`2` HARPOON、`3` MK-45 II、`4` RADAR
- キャンセル: Escape
- ミュート: `M`

## ゲームルール

- 自軍・敵軍とも戦艦5マス、駆逐艦3マス、潜水艦1マス
- 交互に1行動。全艦撃沈で勝利、自軍全滅で敗北
- MISS周囲8マスに生存艦があれば ECHO を表示
- HARPOON: 3×3の四隅と中央を1回攻撃
- MK-45 II: 任意の異なる2地点を1回攻撃
- SPS-10 RADAR: 2×2範囲を3回索敵（ダメージなし）
- 搭載艦が撃沈されると、その特殊兵装は残数があっても使用不能

艦影・画面・効果音・BGMは既存作品の素材を使わず、Canvas 2D / CSS / Web Audio APIで生成しています。
