# DEEP BLUE GRID

公開版: https://spthx.github.io/deep-blue-grid/

ゲームボーイ時代のレーダー作戦ゲームを現代的に再構成した、1人用・全6ステージのブラウザ海戦ゲームです。8×8の海域へ艦隊を配置し、通常砲撃・特殊兵装・レーダーを使って敵AI艦隊の全撃沈を目指します。

## 起動方法

Node.js 22.13 以上を用意し、プロジェクトのフォルダで次を実行します。

```bash
npm install
npm run dev
```

ターミナルに表示されたURLをPCまたはスマートフォンのブラウザで開いてください。同じWi-Fi内のスマートフォンで試す場合は、必要に応じて `npm run dev -- --host 0.0.0.0` を使います。

本番ビルドと自動テスト:

```bash
npm run build
npm test
```

GitHub Pages用の静的ビルドは `npm run build:pages` で生成できます。`master` ブランチへのpush時はGitHub Actionsが自動公開します。

## 操作

- 配置: 艦艇カードを選び海図をクリックまたはタップ
- 回転: 右クリック、または `R`
- 照準移動: 矢印キー / WASD / ポインター
- 決定: Enter / Space / タップ
- コマンド: `1` FIRE、`2` HARPOON、`3` MK-45 II、`4` RADAR
- キャンセル: Escape
- ミュート: `M`

## ゲームルール

- CASUALは公開情報で戦う標準モード
- TACTICSは敵先攻・敵損傷秘匿・判断強化
- SURVIVALはTACTICS相当の敵に全6隻で挑み、撃沈艦と搭載兵装が以降のステージから永久に失われる
- SURVIVALの生存艦はステージクリア時に完全修復され、兵装使用回数も全回復する
- SURVIVALのリトライは、そのステージへ到達した時点の残存艦隊で再配置してやり直す
- TACTICSとSURVIVALでは敵艦種も未識別となり、各艦に1マスある重要区画へ命中すると艦種とコードを識別できる
- 自軍の重要区画は菱形で常時表示され、敵に命中されると敵AIもその艦種を識別する（追加ダメージなし）
- 交互に1行動。全艦撃沈で勝利、自軍全滅で敗北
- MISS周囲8マスに生存艦があれば ECHO を表示
- HARPOON: 3×3の四隅と中央を1回攻撃
- MK-45 II: 任意の異なる2地点を1回攻撃
- SPS-10 RADAR: 2×2範囲を2回索敵（CONTACTは黄、CLEARは青緑。ダメージなし）
- 搭載艦が撃沈されると、その特殊兵装は残数があっても使用不能

## 素材の出所

- 盤面、艦影、波、命中表示などのゲーム画面は、このプロジェクト用のCanvas 2D / CSSコードで描画しています。既存ゲームの画像、スクリーンショット、スプライトは収録していません。
- 効果音とBGMはWeb Audio APIのオシレーターで実行時に合成しています。音声ファイルや既存楽曲のサンプルは収録していません。
- フォントは端末に入っている `Courier New`、`Yu Gothic UI`、`monospace` をCSSで指定するだけで、フォントファイルは配布していません。
- `public/og.png` は、このゲームの告知画像としてOpenAIの画像生成機能で作成したものです。既存ゲームの画像は入力・合成していません。
- React、Next.js、Viteなどの外部ソフトウェアは、それぞれのオープンソースライセンスに従います。
